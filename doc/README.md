# MiniApiServer Documentation

Questa directory raccoglie la documentazione tecnica della solution.

Documenti disponibili:

- `solution-overview.md`: panoramica della solution, dei layer e del flusso applicativo.
- `type-map.md`: catalogo sintetico dei progetti e delle responsabilità principali di classi, interfacce e record pubblici.

La documentazione è allineata con i commenti XML inseriti nel codice sorgente dei progetti in `src/`.
